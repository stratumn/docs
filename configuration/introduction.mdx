## Workflow

{/* A configuration project is a project that implements the business logic of a Stratumn workflow. */}
In Stratumn, a workflow correspond to a business process. It defines a set of actions that are done in a specific order, automatically or manually by users.

### Trace

A trace is an instance of a workflow. It is created when the first action of the workflow is made, and contains all data related to the execution of the workflow.
